# NexaServices — Site vitrine professionnel

NexaServices est un site vitrine moderne et premium destiné à présenter des services professionnels pour les entreprises et les particuliers.

Le site met en avant une image sérieuse, rassurante et haut de gamme grâce à une interface claire, des cartes de services, des packs tarifaires, des témoignages clients et un formulaire de contact.

## Aperçu du projet

Le site présente une agence multi-services proposant notamment :

- Nettoyage professionnel
- Livraison & logistique
- Formation
- Assistance administrative
- Maintenance & support
- Services personnalisés

L’objectif du site est de convaincre rapidement le visiteur, de présenter les prestations de manière claire et de l’encourager à demander un devis.

## Sections du site

### 1. Accueil

La section d’accueil présente un hero premium avec :

- un titre principal fort ;
- une description courte ;
- deux boutons d’action ;
- une image professionnelle ;
- des statistiques de confiance.

### 2. Services

La section services présente les prestations sous forme de cartes visuelles.  
Chaque carte contient :

- une image représentative ;
- une icône ;
- un titre ;
- une courte description ;
- un lien vers le contact.

### 3. À propos

Cette section explique pourquoi choisir NexaServices :

- rapidité d’intervention ;
- fiabilité ;
- accompagnement personnalisé.

### 4. Packs

Le site propose trois packs :

- Pack Starter
- Pack Pro
- Pack Premium

Les prix doivent idéalement rester formulés avec “à partir de” ou “sur devis”, car les services peuvent varier selon le besoin du client.

### 5. Avis clients

La section avis permet de renforcer la crédibilité du site avec des témoignages, des étoiles et des profils clients.

### 6. Contact

La section contact contient :

- un formulaire de demande de devis ;
- un numéro de téléphone ;
- une adresse email ;
- une adresse ;
- un bouton WhatsApp.

## Installation

Aucune installation complexe n’est nécessaire.

1. Télécharger le projet.
2. Ouvrir `index.html` dans un navigateur.
3. Modifier le contenu dans `index.html`.
4. Modifier les styles dans `style.css`.
5. Modifier les interactions dans `script.js`.

## Personnalisation

### Modifier les informations de contact

Dans `index.html`, remplacer :

```html
+212 6 00 00 00 00
contact@nexaservices.ma
Casablanca, Maroc
```

par les vraies informations du client.

### Modifier le lien WhatsApp

Remplacer le numéro dans le lien :

```html
https://wa.me/212600000000
```

par le vrai numéro WhatsApp du client.

### Modifier les images

Il est recommandé de télécharger les images et de les placer dans le dossier `images/`, puis d’utiliser des chemins locaux :

```html
<img src="images/nettoyage.jpg" alt="Nettoyage professionnel">
```

Cela rend le site plus stable et plus professionnel.

## Conseils d’amélioration

Pour rendre le site plus convaincant :

- utiliser des images réelles avec des personnes en action ;
- éviter les images trop génériques ;
- supprimer les sections répétitives ;
- garder les cartes compactes ;
- éviter les promesses trop fortes comme “services illimités” ou “24/7” si elles ne sont pas garanties ;
- préférer “à partir de” ou “sur devis” pour les prix variables.
